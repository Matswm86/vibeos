VibeOS
Vibbey is watching the installer — pick wisely.
Step 1

Welcome

Pick your language. This only affects the installer — you can change the system language later in Plasma's settings.

Requirements: 15 GB free disk, 2 GB RAM, plugged in. Calamares will warn if any are missing.
Step 2

Location

Pick your region (usually auto-detected from your IP). This sets the timezone — the clock on the installed system will be correct from the first boot.

Step 3

Keyboard

Pick your layout. Norwegian users: Norwegian. US/UK users: default. Test the layout by typing a few characters in the preview field.

Step 4

Partitions

This is the screen to take slowly. Three choices:

Erase disk — clean install

Pick this if the drive should be VibeOS only. Every partition is wiped. Do not pick this if there's anything on the drive you want to keep.

Install alongside — dual-boot

Pick this if you want to keep Windows, macOS, or another Linux on the same drive. Calamares shrinks the existing OS partition and installs VibeOS next to it. At boot, you'll get a menu to pick which OS to run.

Always back up before shrinking. Shrinking is safe in 99% of cases — the 1% is bad sectors or a nearly-full source partition.
Manual partitioning — expert

Pick this if you want to reuse an existing Linux partition, split your disk by hand, or add separate /home or encrypted partitions. If you're unsure, don't pick this.

File system: VibeOS uses ext4. EFI partition: 512 MB, auto-created. Swap: a swap file (enables suspend-to-disk on laptops).
Step 5

Create your user

Pick a name, username, password, and hostname (the name this computer announces on the network). The password protects your login and sudo.

Tip: VibeOS auto-logs you in if you tick "Log in automatically." For a laptop you carry around, leave that unchecked.
Step 6

Summary — last chance

Calamares shows exactly which device and partitions will be changed. Read it. If anything looks wrong, click Back and fix it. Hitting Install starts writing to disk — at that point the previous partition table is gone.

Installing

Writing VibeOS to disk

Filesystems get formatted, files copy across, GRUB is installed. Five to ten minutes on an SSD. You can minimize Vibbey and watch the Calamares progress bar — both work.

When it's done, click Restart now, remove the USB stick when the screen goes black, and boot into your new VibeOS.

First login after install: Vibbey comes back in chat mode with qwen2.5:3b loaded locally.
