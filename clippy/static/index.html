About Vibbey

Vibbey is the VibeOS onboarding assistant. She runs 100% locally via Ollama (gemma3:4b by default). She is not Claude. Claude Code takes over once you’re set up.

3D model: Rigged Microsoft Clippy/Clippit by Freedumbanimates on Sketchfab, used under the Sketchfab Standard license.

VibeOS — free, open source, and a little bit fun.
